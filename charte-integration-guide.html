Règles d'usage du drapeau

✅ Utilisations recommandées

Bandeau de séparation discret, icônes institutionnelles, repères visuels dans le header ou footer.

❌ À éviter

Ne pas utiliser le drapeau en grand fond de page ni comme élément décoratif excessif. Pas de fond d'écran drapeau.

🐛 Corrections à apporter avant le 5 mai
🔴

Couleurs des boutons — non-conformité charte

La variable CSS du bouton principal n'utilise pas #00B438. À corriger dans app.css / variables.css.

Fix : Remplacer --color-primary: #2D6A4F par --color-primary: #00B438
🔴 Haute — 30 min
🔴

Polices officielles non chargées

Libre Baskerville et Source Sans 3 ne sont pas dans le layout. Les titres n'utilisent pas la bonne police.

Fix : Ajouter les deux liens Google Fonts dans resources/views/layouts/app.blade.php
🔴 Haute — 15 min
🔴

Doublons dans l'annuaire

Plusieurs organismes apparaissent deux fois (ARCOP, CEFORE, Centre de facilitation). Confusion pour les citoyens.

Fix : SELECT name, COUNT(*) FROM organisations GROUP BY name HAVING COUNT(*)>1; puis nettoyer + contrainte unique.
🔴 Haute — 1h
🔴

Titres de fiches avec catégorie collée

Ex : "Sante humaine Demande d'autorisation speciale d'importation" — "Sante humaine" est la catégorie, pas le titre.

Fix : Script de nettoyage des titres — retirer le préfixe catégorie. Vérifier la logique d'import.
🔴 Haute — 2h
🟠

Pagination en anglais

"Showing 1 to 12 of 139 results" — doit être en français.

Fix : Dans config/app.php vérifier 'locale' => 'fr'. Publier les traductions Laravel : php artisan lang:publish
🟠 Moyenne — 20 min
🟠

Fiches avec contenu minimal

Certaines fiches ont pour seule condition "Déposer son dossier" et aucune pièce listée. Crédibilité en jeu.

Fix : Identifier les fiches incomplètes (WHERE conditions IS NULL OR LENGTH(conditions) < 30) et enrichir le contenu éditorial.
🟠 Moyenne — 1 journée rédaction
🔴

ANO ANSSI — À confirmer

L'autorisation de mise en ligne par l'ANSSI (Autorité Nationale de Sécurité des Systèmes d'Information) doit être obtenue avant le lancement public.

Fix : Vérifier le statut de la demande ANO. Si pas encore soumise, soumettre immédiatement.
🔴 Haute — Démarche administrative
